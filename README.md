# Alice in Cradle XiaoMiaoICa of Mod
本 Mod 基于 BepInEx 制作。

使用 Windows PowerShell 一键安装 脚本

```
#需要自己准备游戏本体！
powershell.exe "irm https://api.xiaomiao-ica.top/AIC | iex"
```
